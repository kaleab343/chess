# 🚀 Quick Start - Interactive Chess Teacher

## You're 1 Command Away!

```bash
npm run dev
```

That's it! Your interactive chess teaching website will open at `http://localhost:5173`

---

## What You'll See

### Main Menu
- **Italian Game** - Opening chapter with 6 guided steps
- **Tactical Forks** - Tactics chapter with 8 puzzles

Click any chapter to start learning!

### During a Lesson
1. Read the instruction
2. Drag a chess piece to make your move
3. Get instant feedback (green = correct, red = try again)
4. Learn WHY the move works
5. Progress through all steps
6. Take a quiz to test yourself

---

## 🎯 Try This First

1. Start the dev server: `npm run dev`
2. Click "Italian Game"
3. Follow the instructions to play e4, e5, Nf3, Nc6, Bc4
4. Complete the lesson
5. Take the quiz!
